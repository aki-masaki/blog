math: oriented segments

2025-12-15

i wanted to learn vectors, but to learn them i must learn oriented segments first...again, i'll use my 9th grade textbook as a guide to learn them

note: on wikipedia i found different notations than the ones in my textbook, for this post im gonna use the ones from my textbook

if we have two points in the same plane, we can define the following:

  • a straight line AB or BA
  • a closed line segment [AB] or [BA] (includes A and B)
  • an open line segment (AB) or (BA) (excludes A and B)
  • the distance between A and B (a real number, noted AB)

the segment [AB] can be traversed both ways, from A to B and from B to A, because of that we introduce oriented or directed line segments which are noted as: \overline{AB} (from A to B) and \overline{BA} (from B to A)

for \overline{AB}, A is called the origin and B is called the extremity (or head and tail)

if A=B then we have \overline{AB}=\overline{AA} which we call a null segment. if A\not=B we call the segment a non-null segment

for oriented segments, we can give the following definitions:

  • the length of \overline{AB} is equal to the length of [AB] and is noted as |\overline{AB}| or, simply AB. |\overline{AA}|=0 (the length of a null segment is always 0)

  • the supporting line for \overline{AB} is AB

  • we can say that two oriented segments have the same direction if their supporting lines either are parraler or are equal

    from definition, a null oriented segment has the same direction as any other oriented segment, since it only has one point, there are infinite lines that can go through that point

  • we can say that \overline{AB} and \overline{A^{\prime}B^{\prime}} have the same sense if they satisfy the following:

    • have the same direction
    • if AB\not=A^{\prime}B^{\prime} then B and B^{\prime} are in the same semi-plane as AA^{\prime}
    • if AB=A^{\prime}B^{\prime}, then the semi-line [A^{\prime}B^{\prime} is included in the semi-line [AB (or inverse)

  • we will say that two oriented segments have opposite sense if they have the same direction but not the same sense

equipollence

what a word...have a look at the dictionary

but basically, two oriented segments are equipollent if they have the same length and sense. we use the tilde (~) to denote that: \overline{AB}\text{\textasciitilde}\overline{CD}

next we're gonna look at three properties:

  • let A, B, C and D be points
    1. \overline{AB}\text{\textasciitilde}\overline{CD}\iff ABCD is a parallelogram or A, B, C, D are on the same line and the segments [AD], [BC] have the same middle
    2. \overline{AB}\text{\textasciitilde}\overline{CD}\iff\overline{AC}\text{\textasciitilde}\overline{BC}
  • basic properties of equivalence
    1. \overline{AB}\text{\textasciitilde}\overline{AB}
    2. \overline{AB}\text{\textasciitilde}\overline{CD}\Rightarrow\overline{CD}\text{\textasciitilde}\overline{AB}
    3. \overline{AB}\text{\textasciitilde}\overline{CD} and \overline{CD}\text{\textasciitilde}\overline{EF}\Rightarrow\overline{AB}\text{\textasciitilde}\overline{EF}
  • let \overline{AB} be an oriented segment. for every point M inside the same plane, there exists only one point N in the same plane such that \overline{MN}\text{\textasciitilde}\overline{AB}
