if f(x):\N^*\rightarrow E then f is the sequence of the set E

the rank of an element its basically its index in the sequence

sequences are commonly notated as (a_n)_n or simply (a_n). to get an element from the sequence we "index" it like this: a_n (thats like saying a[n]) where n\in\N^* is the rank of the number. we can use any other letter

note: unlike in sets, numbers can appear multiple times in a sequence: 1, 1, 2, 3, 4 is a valid sequence

indexing can also start from 0, for example mapping \N to its successor in \N^* (0 \rightarrow 1; 1 \rightarrow 2; 2 \rightarrow 3; ...)

sequences can be defined:

  • in words

d_1=1, d_2=11, d_3=111, ...

here, (d_n)_n is defined as "each element in the sequence has n digits of 1"

  • with a formula

b_n=2^n

that gives: b_1=2^1=2, b_2=2^2=4, b_3=2^3=8, ...

  • using recursion

in sequences, recursion means each element is given by a formula that uses the previous elements

for example, let (c_n)_n be a sequence in which c_1=1, c_2=2, c_{n+2}=c_n+c_{n+1}, for n\geqslant 1

c_3=c_{1+2} (in which case n=1) so c_{1+2}=c_1+c_{1+1}=c_1+c_2=1+2 \Rightarrow c_3=3

a sequence is bound if there exists an interval [a, b] such that a \leqslant a_n \leqslant b

an example of a bound sequence is (\frac{1}{n})_n, n \geqslant 1 because all values fall in the interval [0, 1]

sequences can be increasing if a_n \leqslant a_{n+1} and decreasing if a_n \geqslant a_{n+1}. if a sequence is either increasing or decreasing, it is called a monotone sequence. if we do not allow repeated values (we use < instead of \leqslant and > instead of \geqslant) it is called strictly increasing or strictly decreasing.

there are two special types of sequences called arithmetic sequences (or progressions) and geometric sequences (or progressions). arithmetic sequences just means the next element is the previous element plus a number. geoemtric sequences are the same thing but instead of adding a number, we multiply by one

arithmetic sequences are defined like this: a_{k+1}=a_k+r, k \geqslant 1

r is constant and is called common difference

from the definition, a_{k+1}-a_k=r

for arithmetic sequences we only need the first element and the common difference, and from that we can compute every number in the sequence

for arithmetic sequences, we have some interesting properties

  • mean

for n \geqslant 2 we have a_n=\frac{a_{n-1}+a_{n+1}}{2}

basically, every element n starting from the second, it is equal to the mean of its neighboring elements (a_{n-1} and a_{n+1})

  • formula for general term

in a arithmetic sequence, a_n=a_1+(n-1)r

for a_2=a_1+r, a_3=a_2+r=a_1+r+r, ... this means that an element n is kr far from an element n - k (how far means a_n-a_{n-k}=kr), and to get how far it is from 0 we just add a_1

now for geometric sequences, a_{k+1}=a_k \cdot q

q\not=0 is constant and is called common ratio

from the definition, \frac{a_{k+1}}{a_k}=q

just as with arithmetic sequences, in geometric sequences you only need the first element and the common ratio to compute any element n

  • geometric mean

a_n=\sqrt{a_{n-1} \cdot a_{n+1}}

basically, every element n starting from the second, is equal to the geometric mean of its neighboring elements (a_{n-1} and a_{n+1})

  • formula for general term

a_n=a_1 \cdot q

for a_2=a_1 \cdot q, a_3=a_2 \cdot q=a_1 \cdot q \cdot q therefore a_3=a_1 \cdot q^2 so each element n is q^k far from n-k (how far means a_n-a_{n-k}=q^k)

that's all for sequences!
